# 2. Présentation du Projet

Ce projet consiste en la réalisation d’un site vitrine moderne du département informatique de l’EFREI.

Le site présente :
- les formations proposées,
- l’équipe enseignante,
- les projets pédagogiques,
- les informations de contact,
- les admissions,
- la vie étudiante,
- les spécialisations du département.

Le site a été conçu dans un objectif pédagogique dans le cadre du module TI402 de programmation web.

---

# 3. Fonctionnalités Principales

## Site Responsive
Le site s’adapte aux :
- ordinateurs,
- tablettes,
- téléphones mobiles.

## Carrousel Dynamique
Carrousel interactif avec :
- images du campus,
- étudiants,
- cybersécurité,
- environnement technologique.

## Recherche Dynamique
Recherche instantanée dans les formations grâce à JavaScript.

## Affichage Dynamique JSON
Chargement automatique des enseignants depuis un fichier JSON via Fetch API.

## Formulaire de Contact
Formulaire interactif avec :
- validation des champs,
- vérification email,
- compteur de caractères,
- fenêtre de confirmation.

## Chatbot JSON
Assistant virtuel capable de répondre à plusieurs questions :
- formations,
- équipe,
- web,
- cybersécurité,
- admissions,
- technologies.

## Animations JavaScript
- animations d’apparition,
- statistiques animées,
- transitions visuelles.
